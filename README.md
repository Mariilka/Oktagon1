# Oktagon1
# REST API для каталога книг и категорий

Лабораторная работа по интеграции базы данных PostgreSQL, SQLAlchemy и фреймворка FastAPI. Реализована чистая многоуровневая архитектура с валидацией бизнес-логики.

## Как запустить проект локально

1. **Активируйте виртуальное окружение** в терминале WSL:
   ```bash
   source venv/bin/activate
   ```
2. **Установите зависимости** (если они не установлены):
   ```bash
   pip install -r requirements.txt
   ```
3. **Запустите веб-сервер** Uvicorn:
   ```bash
   uvicorn app.main:app --reload
   ```

##  Проверка и тестирование

* **Интерактивная документация (Swagger UI)**: доступна по адресу `http://127.0.0`
* **Проверка жизнеспособности (Health Check)**: доступна по адресу `http://127.0.0` (автоматически проверяет статус подключения к СУБД PostgreSQL).

*Все скриншоты успешного тестирования (Swagger UI и cURL-запросы с кодом 200) сохранены в директории `examples/`.*
